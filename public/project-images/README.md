# Project Images

Place your project screenshots/images here with these filenames:

- `reganbyte.jpg` (or .png, .webp)
- `convo-note.jpg`
- `founder-dataroom.jpg`
- `lenziq.jpg`
- `phantomstrike.jpg`
- `orby-pos.jpg`

## Recommended Image Specifications:
- **Format:** JPG, PNG, or WebP
- **Size:** 1200x800px or similar (16:9 aspect ratio works best)
- **File Size:** Optimize to under 500KB for faster loading
- **Naming:** Use lowercase with hyphens (e.g., `reganbyte.jpg`)

## Image Optimization Tips:
- Use tools like TinyPNG, Squoosh, or ImageOptim to compress
- WebP format provides best compression
- Keep aspect ratio consistent across all images
